Description:
This project is a gym center where people can register and find out the description of the gym and trainers. They can also see how to work some moves like pushups,squats and shoulder press.

Technologies used:
-HTML
-CSS
-JavaScript

How to use:
1. You open the website and your instantly met by a navigation bar that can help you go to different sections easily.
2. You can scroll and check out the info about the gym and trainers.
3. In the moves place, each move has a timer and three buttons.You click the play button to see how the move is done and the timer starts.The pause button pauses both the timer and the movement.The reset button pauses the movement and returns the time to it's origin.
4. In the membership section u can choose your prefered membership but once you subscribe you can't unsubscribe or downgrade.(you are allowed to upgrade)
5. You can also see the autoscrolling caroussel which will change feedback every 5 secs.
6. Finally in the contact us section you can write a message by entering your name,email and message and submitting it.

Main features:
-Nav bar that makes it easier to go through the website
-A timer and animation for the moves section
-A membership plan with working subscriptions
-Autoscrolling carousel for feedback
-A form you can fill out for any message
-A footer that works like the navbar
